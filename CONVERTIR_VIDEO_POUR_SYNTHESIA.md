# Convertir votre vidéo pour Synthesia

Synthesia accepte principalement **MP4** et **WebM**. Les vidéos sortant de l’app Photos peuvent être en MOV ou HEVC, ce qui pose souvent problème.

---

## Méthode 1 : Exporter depuis Photos (essai rapide)

1. Ouvrez **Photos** sur votre Mac.
2. Sélectionnez la vidéo.
3. Menu **Fichier** → **Exporter** → **Exporter 1 vidéo**.
4. Dans la fenêtre :
   - Choisir **Format vidéo** : **MP4** (si disponible) ou **Vidéo la plus compatible**.
   - Cliquez sur **Exporter**.
5. Enregistrez le fichier sur le **Bureau** ou dans **Téléchargements**.
6. Réessayez l’import dans Synthesia avec ce fichier.

---

## Méthode 2 : Convertir avec QuickTime (Mac)

Si Photos exporte en MOV :

1. Ouvrez la vidéo avec **QuickTime Player** (double-clic sur le fichier).
2. Menu **Fichier** → **Exporter vers** → **Vidéo uniquement**.
3. Choisir **1080p** ou **720p** si nécessaire.
4. Enregistrez (souvent en MOV).

QuickTime peut ne pas donner de MP4. Dans ce cas, utilisez la méthode 3.

---

## Méthode 3 : Convertir en MP4 avec un outil gratuit

### Option A : CloudConvert (en ligne)

1. Allez sur **https://cloudconvert.com**.
2. Cliquez sur **Select File** et choisissez votre vidéo.
3. Choisissez le format de sortie **MP4**.
4. Cliquez sur **Convert**.
5. Téléchargez le fichier MP4.
6. Importez ce fichier dans Synthesia.

### Option B : HandBrake (app Mac, gratuit)

1. Téléchargez **HandBrake** : https://handbrake.fr.
2. Installez et ouvrez HandBrake.
3. Cliquez sur **Open Source** et sélectionnez votre vidéo (ou glissez-déposez).
4. Dans **Preset**, choisissez par ex. **Fast 1080p30**.
5. Onglet **Summary** : vérifier que le format est **MP4**.
6. Cliquez sur **Start**.
7. Importez le MP4 obtenu dans Synthesia.

---

## Méthode 4 : Copier la vidéo depuis Photos

1. Dans **Photos**, sélectionnez la vidéo.
2. Glissez-déposez-la sur le **Bureau** ou dans un dossier.
3. Regardez l’extension du fichier créé (`.mov`, `.mp4`, etc.).
4. Si c’est `.mov`, utilisez CloudConvert ou HandBrake pour le convertir en MP4.

---

## Contraintes Synthesia

- **Formats acceptés** : MP4, WebM.
- **Taille max** : 250 Mo.
- **Conseil** : privilégier MP4 (codec H.264).

---

## Si ça bloque encore

Essayez aussi :

1. **Réduire la qualité** : exporter en 720p au lieu de 1080p/4K.
2. **Réduire la durée** : si la vidéo est très longue, la couper à 30–60 s pour l’import.
3. **Tester avec une capture courte** : enregistrer 10–20 s de démo avec QuickTime, puis l’importer dans Synthesia.
