# Guide pas à pas : créer votre vidéo marketing avec l’IA

**Objectif :** Une vidéo d’environ 1 minute pour présenter la plateforme aux garderies, avec voix IA.

---

## MÉTHODE 1 : Synthesia (recommandé)

### Étape 1 — Créer un compte
1. Allez sur **https://www.synthesia.io**
2. Cliquez sur **Get started for free**
3. Inscrivez-vous avec votre email ou Google
4. Le compte gratuit offre environ 36 minutes de vidéo par an

### Étape 2 — Nouvelle vidéo
1. Dans le tableau de bord, cliquez sur **Create new video**
2. Choisissez **Blank video** (vidéo vierge)
3. Donnez un titre : par ex. « Gestion Vacances – Démo »

### Étape 3 — Choisir un avatar
1. Cliquez sur le bloc **Avatar** à gauche
2. Parcourir les avatars ou utiliser le filtre **French**
3. Choisir un avatar professionnel (ex. « Anna », « Marc »)
4. Choisir un fond simple (bureau ou neutre)

### Étape 4 — Coller le script
1. Dans la zone de texte, collez le script ci-dessous
2. Vérifier que le texte est correct
3. Ajuster si nécessaire (Synthesia lit le script automatiquement)

### Étape 5 — Ajouter les captures d’écran de la plateforme
1. Cliquez sur **+** ou **Add element** pour ajouter une scène
2. Choisir **Image** ou **Screen recording**
3. **Avant d’enregistrer :**
   - Ouvrir la plateforme sur Railway (ou localhost)
   - Prendre des captures (Cmd+Shift+4 sur Mac) ou enregistrer l’écran
4. **Scènes à inclure :**
   - Page de connexion (choix des profils)
   - Dashboard éducatrice avec une demande
   - Interface admin avec les demandes
5. Insérer ces images/vidéos entre les séquences de l’avatar

### Étape 6 — Ajuster le timing
1. Répartir le script sur environ 60 secondes
2. Placer les images aux bons moments (ex. « L’éducatrice choisit son profil » = image page connexion)
3. Vérifier la durée totale

### Étape 7 — Générer la vidéo
1. Cliquer sur **Generate**
2. Attendre 5 à 15 minutes
3. Prévisualiser le résultat
4. Télécharger en MP4

## MÉTHODE 2 : HeyGen (alternative)

### Étape 1 — Créer un compte
1. Allez sur **https://www.heygen.com**
2. Cliquez sur **Sign up free**
3. Inscrivez-vous avec email ou Google

### Étape 2 — Créer une vidéo
1. Cliquez sur **Create Video**
2. Choisir **AI Studio** ou **Template**
3. Pour une démo produit : **AI Studio**

### Étape 3 — Script et avatar
1. Coller le script dans la zone de texte
2. Choisir un avatar (filtre French si besoin)
3. Sélectionner une voix française naturelle

### Étape 4 — Ajouter du contenu
1. Utiliser **Add scene** pour insérer des captures d’écran
2. Importer les images de la plateforme
3. Synchroniser avec le script

### Étape 5 — Générer
1. Cliquer sur **Submit** ou **Generate**
2. Télécharger la vidéo une fois prête

## SCRIPT PRÊT À COPIER-COLLER

```
Gérez les congés de votre garderie en quelques clics. Simple, transparent, sans papier.

L'éducatrice choisit son profil, entre son mot de passe, et accède à son tableau de bord. Elle crée sa demande en 10 secondes. Réponse immédiate ou examen par l'administration.

L'administration supervise tout depuis une seule interface. Règles flexibles : dates interdites, ratio qualifiées, ancienneté. Tout est personnalisable.

Première arrivée, première servie. Pas d'email, pas de papier. Urgences motivées traitées directement sur la plateforme.

Gestion Vacances — Les Amis Bout De Choux. Pour les garderies qui veulent simplifier la gestion des congés.
```

## PRÉPARATION DES VISUELS (avant de lancer Synthesia/HeyGen)

### À enregistrer ou capturer
1. **Capture 1** : Page d’accueil avec les profils (0–5 s)
2. **Capture 2** : Dashboard éducatrice, formulaire « Nouvelle demande » (5–20 s)
3. **Capture 3** : Liste des demandes avec statuts (10–15 s)
4. **Capture 4** : Interface admin, onglet Demandes (20–35 s)
5. **Capture 5** : Onglet Règles ou Éducatrices (35–45 s)

### Outils de capture
- **Mac** : Cmd+Shift+4 (capture) ou QuickTime (enregistrement écran)
- **Windows** : Win+Shift+S ou Xbox Game Bar

### Conseil
Enregistrer 30–40 secondes d’écran en naviguant sur la plateforme, puis insérer cet extrait dans la vidéo IA pour montrer le produit en action.
